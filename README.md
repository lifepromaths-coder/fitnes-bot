# 🏋 Фитнес-бот для Telegram

Бот считает калории по тексту и фото, ведёт дневник питания и тренировок.

## Что умеет
- 📸 Фото еды → определяет блюдо и считает калории
- ✍️ Текст ("съел гречку 200г") → считает калории и БЖУ
- 💪 Тренировки → логирует и хвалит
- /stats → статистика за день
- /week → итог за неделю
- /reset → сброс дня

---

## Способ 1: Запуск на Railway (БЕСПЛАТНО, рекомендуется)

### Шаг 1 — Зарегистрируйтесь на Railway
1. Зайдите на https://railway.app
2. Войдите через GitHub (создайте аккаунт GitHub если нет)

### Шаг 2 — Создайте проект
1. Нажмите "New Project"
2. Выберите "Deploy from GitHub repo"
3. Загрузите файлы бота (или создайте репозиторий на GitHub)

### Шаг 3 — Добавьте переменные окружения
В настройках проекта → Variables добавьте:
```
TELEGRAM_TOKEN = ваш_токен_от_botfather
ANTHROPIC_API_KEY = ваш_ключ_sk-ant-...
```

### Шаг 4 — Deploy
Railway сам установит зависимости и запустит бота.

---

## Способ 2: Запуск на своём компьютере (для теста)

### Требования
- Python 3.10+
- pip

### Установка

```bash
# Установите зависимости
pip install -r requirements.txt

# Создайте файл .env
cp .env.example .env
# Откройте .env и вставьте ваши токены

# Запустите бота
python bot.py
```

### Файл .env должен выглядеть так:
```
TELEGRAM_TOKEN=8814209714:ВАШ_РЕАЛЬНЫЙ_ТОКЕН
ANTHROPIC_API_KEY=sk-ant-api03-ВАШ_РЕАЛЬНЫЙ_КЛЮЧ
```

## Способ 3: Docker

```bash
docker build -t fitness-bot .
docker run -e TELEGRAM_TOKEN=ваш_токен -e ANTHROPIC_API_KEY=ваш_ключ fitness-bot
```

## Как пользоваться ботом

После запуска найдите бота в Telegram: @family_ni_tracker_bot

Примеры сообщений:
- "Съел овсянку 150г с молоком и 2 яйца"
- "Обед: гречка 200г, куриная грудь 150г, огурец"
- [Отправить фото тарелки]
